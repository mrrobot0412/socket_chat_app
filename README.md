
# Socket Chat App

A real-time chat application built with the MERN stack and Socket.io.
Features secure user authentication, real-time messaging, group chats, and notifications with encrypted data storage in MongoDB.
## Tech Stack

**Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB

## Run Locally

Clone the project

```bash
  git clone https://github.com/mrrobot0412/socket_chat_app
```

Go to the project directory

```bash
  cd socket_chat_app
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the server

```bash
  npm run start
```
Start the Client

```bash
  //open now terminal
  cd frontend
  npm start
```
